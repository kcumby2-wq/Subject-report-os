# Monthly Workflow — Report Generation + Goal Review

1. Run the daily workflow first.
2. Generate the full monthly report per `skills/reporting/SKILL.md` covering all 6 goals in `goals/goals.md`.
3. For each goal: assess progress against target, decide if it's on track, at risk, or complete.
4. Move any completed goals to `goals/archive/` with outcome notes.
5. Update `goals/goals.md` review date and propose any goal/target changes to Kyron for approval.
6. Save the monthly report to `output/reports/` with the date in the filename.
