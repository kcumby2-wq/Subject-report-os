# Weekly Workflow — Monday Planning + Friday Review

## Monday Planning

1. Run the daily workflow (feedback check, SLA guard, data intake, goals pulse).
2. Review the week ahead: known transcript orders due, program cohort milestones, any competitions/events on the calendar.
3. Decide the production focus for each day of the week (content / course / nurture / report / upsell — see `workflows/daily.md` Step 5).
4. Note any blockers that need Kyron's input before the week starts.

## Friday Review

1. Run the daily workflow first.
2. Pull the week's numbers against `goals/goals.md`: transcripts delivered, SLA hit rate, new members/churn, course completion movement, Active Profile pipeline movement, SM upsell performance.
3. Write the weekly report per `skills/reporting/SKILL.md` and save to `output/reports/`.
4. Flag anything that should roll into next Monday's planning.
